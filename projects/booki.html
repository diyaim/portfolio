Booki

Intégration HTML/CSS responsive d’une page à partir de maquettes Figma.

Retour aux projets Code GitHub

Contexte

Booki est une start-up souhaitant proposer un site permettant aux utilisateurs de trouver des hébergements et des activités dans la ville de leur choix.

Ma mission est d’intégrer l’interface en HTML et CSS à partir d’une maquette Figma, en respectant une structure de fichiers imposée (index.html, dossier css, dossier images).

Objectifs

  • Respecter la maquette (structure, espacements, hiérarchie).
  • Assurer un rendu responsive (desktop, tablette, mobile).
  • Produire un HTML sémantique et un CSS clair.
  • Garantir un rendu stable (cartes, images, alignements).

Réalisation

  • Structuration HTML à partir du découpage de la maquette.
  • Intégration du header, du formulaire de recherche et des filtres.
  • Création et duplication des cartes (hébergements, populaires, activités).
  • Mise en place du responsive via media queries (≤1024px, <768px).
  • Vérification via DevTools + validateurs W3C.
